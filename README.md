🌤 Weather App
This is a simple weather application built using HTML, CSS, and JavaScript.
Its a project assignment which was given out by COURSERA and it allows users to search for any city and displays real-time weather data using the OpenWeatherMap API, including temperature, weather conditions, and location.
To app features a modern, centered design with smooth animations and dynamic backgrounds that change based on the weather. To
Touse the project, insert your API key into the JavaScript file, open the index.html file in a browser, and search for a city to view its weather.
